## Overview

Anubis is a Minecraft Firewall Utility that [weighs the soul of your connection](https://en.wikipedia.org/wiki/Weighing_of_souls) using PoW challenges in order to protect upstream resources from bots.

Anubis is as lightweight as possible to ensure that everyone can afford to protect the communities closest to them.

Anubis is configurable, and you can find the configuration files under `config/anubis/`.

Anubis is inspired by [TecharoHQ/anubis](https://github.com/TecharoHQ/anubis) and uses its image material as an icon.
